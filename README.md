# stock-market-data-pipeline
End-to-end stock market data pipeline using API extraction, ETL processing, MySQL database storage, and Power BI dashboard visualization.
<BR>This project demonstrates an end-to-end data pipeline for stock market analytics. The pipeline extracts stock data from an external API, performs ETL transformations, stores the processed data in a SQL database, and visualizes insights using Power BI.

This project highlights data engineering, ETL workflow, and business intelligence skills.
